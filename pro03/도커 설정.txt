웹 사이트 배포에 필요한 환경설정 정보를 지정함.

도커를 실행하면 도커설정 파일에서 지정한 대로 서버 전체를 
온전히 복제한 컨테이너 이미지가 생성됨.

컨테니너 이미지를 작동시키면 컨테이너가 실행 되면서 기존과 동일한
상태로 웹사이트 서버를 실행...

컨테이너 이미지 처음 생성시 서버를 실행하기 위해 필요한 모든 파일은 내려받아야
하기 때문에 시간이 걸림.
두 번째 작업부터 이미 받아놓은 이미지에서 변경된 부분만 갱신, 시간단축.

docker-compose build
docker-compose up -d(백그라운드에서 돌아가도록 만드는 명령어)

docker image ls
docker container ls

docker-compose down(컨테이너 중단)
